# Test

Delegate to the Tester agent to write and/or run tests.

Invoke the Tester with:
- What was implemented (feature or fix)
- Existing test patterns (framework, location, conventions)
- Whether to write tests, run tests, or both

The Tester:
- Writes unit and integration tests matching project conventions
- Runs the test suite and reports results
- Flags any failures with diagnosis

Report to the Director: tests written, tests passing/failing, coverage gaps.
