# Session Start

Read SESSION.md in the project root. Then:

1. Brief the Director with:
   - **Last session:** date and one-line status
   - **In progress:** anything not yet complete
   - **Next steps:** top 3 priority items
   - **Blockers:** any unresolved blockers

2. Ask: "Continue from here, or something new?"

Keep the brief to 5–8 lines. No padding.
