# Session End

Wrap up the session cleanly:

1. **Update SESSION.md:**
   - Add a new `## Completed This Session (YYYY-MM-DD)` block with what was done today
   - Move any completed items out of "In Progress"
   - Update "Next Steps" to reflect what's actually left
   - Clear resolved blockers (write "None" if clear)

2. **Commit:** `git add -A && git commit -m "[type]: description"`
   - Do NOT push — push requires Director confirmation

3. **Report to Director:**
   - What was completed this session (3–5 bullets)
   - What's next (top 2–3 items)
   - Any blockers

Keep it concise. Status first, detail second.
